CPL Airlines - Sistema de Gerenciamento de Passagens

Projeto desenvolvido para a disciplina de Algoritmos e Estrutura de Dados I.

- Objetivo:

Desenvolver um sistema em C para gerenciamento de passagens aéreas, permitindo:

- Cadastro de passageiros
- Visualização de voos disponíveis
- Compra de passagens
- Cancelamento de passagens
- Visualização de assentos
- Listagem de passagens


- Conceitos de AED1 que foram utilizados:

. TAD (tipo abstrato de dado)
. Listas encadeadas
. Alocação dinâmica
. Modularização
. Busca

- Estrutura:

  main.c
  funcoes.h
  funcoes.c

- Como compilar:
  gcc main.c funcoes.c -o programa
